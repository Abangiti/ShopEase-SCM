ShopEase Online Shopping Application
Manual Test Cases

Test Case ID: TC-01
Test Case Name: Valid User Login
Description: Verify that a user can log in with correct credentials.
Steps:
1. Open the login page
2. Enter valid username and password
3. Click Login
Expected Result:
User is redirected to the products page.
Status: Pass


Test Case ID: TC-02
Test Case Name: Invalid User Login
Description: Verify system behavior for invalid login credentials.
Steps:
1. Open the login page
2. Enter invalid username or password
3. Click Login
Expected Result:
Error message is displayed.
Status: Pass


Test Case ID: TC-03
Test Case Name: Product Search
Description: Verify product search functionality.
Steps:
1. Log in to the system
2. Enter a product name in the search box
Expected Result:
Matching products are displayed.
Status: Pass


Test Case ID: TC-04
Test Case Name: Add Product to Cart
Description: Verify that products can be added to the cart.
Steps:
1. Log in
2. Click "Add to Cart" on a product
Expected Result:
Product is added to the cart successfully.
Status: Pass


Test Case ID: TC-05
Test Case Name: View Shopping Cart
Description: Verify cart page functionality.
Steps:
1. Add product to cart
2. Open cart page
Expected Result:
Selected products are displayed in the cart.
Status: Pass
